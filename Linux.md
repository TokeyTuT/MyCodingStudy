# Linux 命令

在 Linux 中，无论什么命令，什么用途，命令都有通用的格式：

`command [-options][parameter]`

*   *conmmad* ：命令本身
*   -*options*：[可选，非必填]，控制命令行为细节
*   *parameter*：[可选，非必填]，控制命令的指向目标



## ls

`ls [-a -l -h] [Linux 路径]`

*   -a ： 查看所有文件，包括隐藏文件 —— **具有前导“.”**
*   -l：以列表的形式展示，注意，

**注意**：可选命令可以联用哦～

Linux 路径就是你要具体查看某个路径下文件夹内容



## cd / pwd

*   `cd [Linux 路径]` ：从当前目录切换到指定的 Linux 路径
*   `pwd` ：告诉使用者当前在哪个路径下

相对路径：以当前目录为起点，描述路径的一种写法，路径描述无需以/开头

绝对路径：以根目录为起点，描述路径的一种写法，路径描述以/开头

### 特殊命令符

*   **.**  —— 当前路径，比如 `cd ./Desktop……`
*   **..**  —— 上一级路径，比如 `cd..`
*   **~** —— 表示 HOME 目录，比如：`cd ~`
*   **-** ——刚刚所处的地方，比如：`cd -`

## mkdir

`mkdir [-p] Linux 路径`

*   Linux 路径必选，表示要创建的文件夹的路径，相对路径或者绝对路径均可
*   -p：可选，自动创建不存在的父级目录适用于连续创建多层级的目录

我们只能在 HOME 路径下添加，应为 Linux 权限很严格。



## touch

`touch Linux 路径`—— 创建空白文件

*   相对路径和绝对路径均可

## cat

`cp [-r]  参数 1 参数 2`

*   参数 1：Linux 路径，表示复制的文件或者文件夹

*   参数 2：Linux 路径，表示要去移动去的地方，如果目标不存在，则进行改名 1，确保目标存在

## rm

直接进抹去在磁盘的内容，不会进回收站

`rm [-r -f -i] Linux 路径`

*   -r ：可选参数，表示递归，如果要删除文件夹必须加上
*   –f：可选参数，强制删除，不提示，即使文件不存在也不报错
*   -i：可选参数，交互式删除，删除前会问你是否删除

## which

**查找某个命令的可执行文件所在路径**

比如说，如果我输入 `which ls`  —— 结果就是 ***/bin/ls***

## find

`find 起始路径 -name “被查找文件名” [执行动作]`

*   按文件名：`find . -name "test.cpp"` —— 在当前目录 (`.`) 找名字叫 `test.cpp` 的文件。
*   按扩展名：`find /home -name "*.java"` —— 在 `/home` 下找所有 Java 源文件。
*   按大小：`find . -size +100M` —— 找出当前路径下所有大于 100MB 的文件
*   按时间： `find . -mtime -1` —— 找出 **24小时内** 修改过的文件（常用于查日志）
*   按类型：`find . -type d` —— 找出所有文件夹（d 代表文件夹，f 代表文件）

进阶神技：搜索并处理 (`-exec`)： 

`find` 不仅能帮你找到文件，还能顺手把活儿干了。比如你写了一堆 C++ 代码，想把所有的 `.out` 编译产物都删掉：

-   **命令**：`find . -name "*.out" -delete`
-   **更通用的方式 (`-exec`)**： `find . -name "*.txt" -exec rm -rf {} \;`
    -   `{}`：代表 `find` 找到的每一个文件。
    -   `\;`：命令的结束标志。



## wc

直接运行 `wc  [可选参数] filename` 时，它会输出三个数字，顺序固定为： **行数** | **单词数** | **字节数** | **文件名**

*   -l ： lines，只统计行数
*   -w：words，只统计单词数
*   -c：bytes，只统计字节数
*   -m：chars，只统计字符数

**注意**：

>   *    **单词的定义**：`wc` 认为“单词”是由空格、制表符（Tab）或换行符分隔的字符串
>   *   **字节 vs 字符**：
>       -   在处理纯英文时，`-c` (bytes) 和 `-m` (chars) 结果一样。
>       -   在处理**中文**时，由于一个汉字在 UTF-8 编码下通常占 3 个字节，所以 `-c` 算出来的数字会比 `-m` 大得多。

## grep

`grep [可选参数] “target” Linux 路径`

在文件（或输入流）中查找符合条件的行，并把它打印出来

| **场景**             | **命令示例**                | **效果**                                            |
| -------------------- | --------------------------- | --------------------------------------------------- |
| **在文件中找词**     | `grep "main" Hello.cpp`     | 找出 `Hello.cpp` 中所有包含 `main` 的行。           |
| **不区分大小写**     | `grep -i "python" data.txt` | 无论写成 Python 还是 python 都能找到。              |
| **在文件夹里递归找** | `grep -r "TODO" ./project`  | 在整个项目文件夹里搜索所有带 "TODO" 标记的代码。    |
| **反向查找**         | `grep -v "error" log.txt`   | 打印所有**不包含** "error" 的行（过滤掉报错信息）。 |





## 管道符 | 

`命令 A | 命令 B`

*   命令 A ： 产生一堆文字信息
*    | ：接住这些信息
*   命令 B：对这些信息进行二次加工

常用的几个组合：

1.   搜索 + 过滤：

     

     >   ```bash
     >   ls | grep "Java"
     >   ```
     >
     >   *   *ls* 把所有文件名吐出来，*grep* 只捡出带有“Java”的字样

2.   查看大型文件：

     >   ```bash
     >   cat hug_data.csv | less
     >   ```
     >
     >   *   *cat* 倒出所有内容，*less* 接住并把它变为**可翻页**模式，空格翻页，按 q 结束

3.    统计 + 计数 ：

      >   ```bash
      >   ps -ef | wc -l
      >   ```
      >
      >   *   `ps - ef`列出所有进程行，`wc -l`数一下一共有多少行

4.    排序 + 去重

      >   ```bash
      >   cat names.txt | sort | uniq | wc -l
      >   ```
      >
      >   *   这是一个“三级流水线”：先读取文件 -> 排序（`uniq` 必须在排序后才生效）-> 去掉相邻重复项 -> 数行数。

**兼容性**：只有能产生“标准输出”（stdout）的命令才能放在左边，只有能接收“标准输入”（stdin）的命令才能放在右边。



## echo 

`echo 输出的内容`

echo 就是在控制台上打印内容，相当于 *print*，，简单内容不用双引号， 复杂可以用双引号包起来。

如果我么想输出当前路径，如果我们直接 `echo pwd`控制台上还是直接 *pwd*，由此我们引出**反引号****``** 包起来就可以了。

## 重定向符号

*   **\>** ：将左侧的结果**覆盖**写入右中： `echo "nihao" > a.txt` —— 如果文件不存在，会直接创建文件

    >   ```bash
    >   ls -a \User\tuttokey > a.txt
    >   ```

*   **\>>** ：将左侧结果追加**写入**右侧中：和上面类似，只是这里不会清空原本文件内容

## tail

`tail [-f -num] Linux 路径`

*   Linux 路径 ： 表示被更随的文件
*   -f： 可选参数，表示持续跟踪
*   -num：可选参数，参看尾部有多少行，不填默认 10 行



## apt

Linux 系统的应用商店，在命令行模式下是 `apt | 命令`

```bash
apt install / remove 程序名[-y]
```

*   *install* ：下载

*   *remove* ：卸载

*   -v：可选命令，不要提示直接执行

注意，在 Linux 中，我们需要用管理员权限才能安装，在指令最前面加上 ***sudo*** —— 表示不切换命令，临时切换管理员权限执行
