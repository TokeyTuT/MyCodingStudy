>   注：这里的写法参考的是国内严蔚敏的 *next*数组写法，下标 0 初始化为-1，《算法导论》中的写法使用一个 $\pi$ 数组

```c++
class Solution {
public:
    int strStr(string haystack, string needle) {
        int m = haystack.size();
        int n = needle.size();

        vector<int> next(n);
        next[0] = -1;
        int i = 0, j = -1; 
        
        while (i < n - 1) {
            if (j == -1 || needle[i] == needle[j]) {
                i++;
                j++;
                next[i] = j;
            } else {
                j = next[j];
            }
        }

        i = j = 0;
        while (i < m && j < n) {
            if (j == -1 || haystack[i] == needle[j]) {
                i++;
                j++;
            } else {
                j = next[j];
            }
        }

        if (j == n) return i - j;
        return -1;
    }
};
```

## 中心扩展法

中心扩展法常常用来解决**回文串**问题。通常，对于一个字符串，如果我们想要找出所有回文子串，如果暴力枚举所有子串，由于有 $O(n^2)$ 个子串，每个子串是否为回文串判断需要$O(n)$的时间，总体复杂度为 $O(n^3)$ ，这太慢了。

中心扩展法可以快速用$O(1)$判断是否为回文串

### 例题

***[最长回文子串](https://leetcode.cn/problems/longest-palindromic-substring/description/)***

>   给你一个字符串 `s`，找到 `s` 中最长的 回文 子串。
>
>    
>
>   **示例 1：**
>
>   ```
>   输入：s = "babad"
>   输出："bab"
>   解释："aba" 同样是符合题意的答案。
>   ```
>
>   **示例 2：**
>
>   ```
>   输入：s = "cbbd"
>   输出："bb"
>   ```
>
>    
>
>   **提示：**
>
>   -   `1 <= s.length <= 1000`
>   -   `s` **仅由数字和英文字母组成**

#### 分析

如果一个子串为 *abcba*，显然它是一个回文子串，对于这个回文子串，它又是由子串*bcb*扩展而来的，而*bcb*又是由子串*c*扩展而来的。

既然如此，为什么不直接从 c 开始向外扩展呢？

*   *c* 是回文串。

*   看看 *c* 左右两边的字母是不是一样的，一样，那么 *bcb* 是回文串。

*   继续，看看 *bcb* 左右两边的字母是不是一样的，一样，那么 *abcba* 是回文串。我们 O(1) 地判断出了一个子串是不是回文串！


这些子串的长度都是奇数，我们称其为**奇回文串**。

回文串还可以是偶数长度，我们称其为**偶回文串**。

比如子串 *abccba*，我们可以从中间的 *cc* 开始：

*   cc 是回文串。
*   看看 cc 左右两边的字母是不是一样的，一样，那么 bccb 是回文串。
*   继续，看看 bccb 左右两边的字母是不是一样的，一样，那么 abccba 是回文串。


一般地，枚举 $i=0,1,2,…,n−1$ 作为奇回文串的中心，向左右两侧扩展：

*   初始化 $l=r=i$。
*   如果 $s[l]=s[r]$，那么向左右两侧扩展，把 l 减一，把 r 加一，继续判断更长的子串是不是回文串。直到下标出界或者 $s[l] \neq s[r]$。
*   循环结束时，最后一轮循环的子串$s[l+1]$ 到 $s[r−1]$ 是回文串，若其长度 $r−l−1$ 大于答案的长度，那么更新答案的左右端点为 $l+1$ 和 $r−1$，方便输出具体子串。

同理，枚举 $i$ 和 $i+1$ 作为偶回文串的中心，也就是初始化 $l=i$，$r=i+1$，其余做法同上。

### 实现

**写法一**：奇偶分开

```c++
class Solution {
public:
    string longestPalindrome(string s) {
        int n = s.size();
        int ans_left = 0,ans_right = 0; //[left,right)

        //奇
        for(int i = 0;i < n;i++){
            int l = i,r = i;
            while(l >= 0 && r < n && s[l] == s[r]){
                l--;
                r++;
            }

            //循环结束时，s[l + 1:r - 1] 都是回文子串
            if(r - l - 1 > ans_right - ans_left){
                ans_left = l + 1;
                ans_right = r;
            }
        }

        //偶    
        for(int i = 0;i < n;i++){
            int l = i,r = i + 1;
            while(l >= 0 && r < n && s[l] == s[r]){
                l--;
                r++;
            }
            if(r - l - 1 > ans_right - ans_left){
                ans_left = l + 1;
                ans_right = r;
            }
        }

        return s.substr(ans_left,ans_right - ans_left);
    }
};
```



**写法二**：合二为一

枚举$i = 0,1,2……,2n - 2$

*   规定当 *i* 是偶数时，使用枚举奇回文串的规则，即初始化$l = r = \frac i 2$。比如$i=2$时 $l=r=1$。
*   规定当 *i* 是奇数时，使用枚举偶回文串的规则，即初始化$l=\lfloor\frac i 2\rfloor$，$r=\lceil\frac{i} 2\rceil$。比如 $i=1$ 时$l=0，r=1$

两种情况可以合并为：

*   初始化$l = \lfloor\frac i 2\rfloor$,$r = \lceil\frac i 2\rceil=$ 

按照这个规则，可以恰好枚举到所有奇偶回文串

```c++
class Solution {
public:
    string longestPalindrome(string s) {
        int n = s.size();
        int ans_left = 0, ans_right = 0;

        for (int i = 0; i < 2 * n - 1; i++) {
            int l = i / 2, r = (i + 1) / 2;
            while (l >= 0 && r < n && s[l] == s[r]) {
                l--;
                r++;
            }
            // 循环结束后，s[l+1] 到 s[r-1] 是回文串
            if (r - l - 1 > ans_right - ans_left) {
                ans_left = l + 1;
                ans_right = r; // 左闭右开区间
            }
        }

        return s.substr(ans_left, ans_right - ans_left);
    }
};
```



*   时间复杂度 $O(n^2)$
*   空间复杂度 $O(1)$

如果想要降低时间复杂度到$O(n)$，可以使用*Manacher 算法*，见下节



### 如何用中心扩展法预处理字符串中所有子回文串呢？

通常来看，如果我们需要在某个算法中$O(1)$ 判断子串*s[l:r]*是否为子回文串，那么我们必须预处理这个字符串，用一个二维数组*is_pal[l\][r\]*储存是否为子回文串，最简单好理解的算法如下：

```python
def pre_operation_str(self,s:str)    
    n = len(s)
    is_pal = [[False for _ in range(n)] for _ in range(n)]

    def expend(l:int,r:int):
        while l >= 0 and r < n and s[l] == s[r]:
            is_pal[l][r] = True
            l-=1
            r+=1

    for i in range(n):
        expend(i,i)
        expend(i,i + 1)
```
