# 心得

我从大一上学期 10 月份左右开始接触算法，当时刚学完 c++所有基本语法，一开始觉得刷算法真痛苦，但越到后面越觉得这东西还是挺有意思的，后面一发不可收拾，短短三个多月就刷了六百多道力扣了，基本保持每天 4～5 道题了，~~但是寒假之后我发现，我压根没有算法天赋，遂放弃~~，现在刷算法纯纯只有无聊的时候才会碰了，不过尽量保持每天都写一道吧）。

这个文件夹内都是我一路刷算法所积累的笔记，虽然就学了半年算法，但其实收获还是颇丰的，也是学完了基本的算法 ：动态规划、简单的图算法、双指针啥的，里面积累的例题还是挺有代表的，不过我现在基本上忘了一半儿了，这东西只要不经常看，基本过几天就忘了，我现在也忘记一半了（~~经典白雪~~）
